## tanimoto係数を求める

```ruby
# a,b は、数値の配列とする

def tanimoto(a,b)
	na=a.sum							# aの１の数
	nb=b.sum							# bの１の数
	azb=a.zip(b) 						# aとbをzipした配列
	ab=azb.map{|x|if x==[1,1] then 1 else 0 end}
	nab=ab.sum						# aとbの共通集合の数
	return nab/(na+nb-nab).to_f		# tanimoto係数
end
```
