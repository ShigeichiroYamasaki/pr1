## tanimoto係数を求める

```
def tanimoto(a,b)
	na=a[(1..-1)].reduce(0){|s,x|s+x}  	# aの１の数
	nb=b[(1..-1)].reduce(0){|s,x|s+x}		# bの１の数
	azb=a.zip(b) 								# aとbをzipした配列
	ab=azb[(1..-1)].map{|x|if x==[1,1] then 1 else 0 end}
	nab=ab.reduce(0){|s,x|s+x}				# aとbの共通集合の数
	return nab.to_f/(na+nb-nab)			# tanimoto係数
end
```
